# mips-assembly

清华大学2024春季学期课程《数字逻辑与处理器基础》大作业一：MIPS汇编语言。

主要任务：将 C++ 代码等价地翻译为 MIPS 汇编代码。

请在 MARS 仿真器中打开 .asm 汇编文件，Windows下中文乱码解决方法：用记事本打开后“另存为”，下面编码改为“ANSI”。在 vscode 中乱码问题有时难以捉摸和解决。
